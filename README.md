# Music Search App

Uma aplicação web simples para buscar músicas usando a API do iTunes.

## Funcionalidades

- Busca por artistas ou músicas
- Reprodução de previews das músicas
- Adicionar/remover músicas aos favoritos
- Armazenamento local dos favoritos

## Como usar

Abra o arquivo `index.html` em um navegador web.

## Tecnologias

- HTML
- CSS
- JavaScript
